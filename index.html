Examples

See EZ in Action

Real code examples to get you started

fibonacci.ez 
// Fibonacci sequence in EZ
task fib(n) {
    when n lessthen 2 {
        give n
    }
    give fib(n - 1) + fib(n - 2)
}

// Print first 10 fibonacci numbers
repeat i = 0 to 10 {
    out fib(i)
}
factorial.ez 
// Calculate factorial
task factorial(n) {
    when n lessthen 2 {
        give 1
    }
    give n * factorial(n - 1)
}

// Get user input and calculate
out "Enter a number: "
in num
out "Factorial: " + factorial(num)
conditions.ez 
// Check if number is positive, negative, or zero
task checkNumber(n) {
    when n greater 0 {
        out "Positive!"
    } other when n less 0 {
        out "Negative!"
    } other {
        out "Zero!"
    }
}

checkNumber(42)   // Output: Positive!
checkNumber(-7)   // Output: Negative!
loops.ez 
// Different loop types in EZ

// For loop - count from 1 to 5
repeat i = 1 to 5 {
    out i
}

// While loop - run until condition is false
x = 10
until x equal 0 {
    out x
    x = x - 1
}

// Foreach - iterate over array
colors = ["red", "green", "blue"]
get color in colors {
    out color
}

Add the folder to your system PATH environment variable:

  1. Press Win + R, type sysdm.cpl, press Enter
  2. Go to Advanced ‚Üí Environment Variables
  3. Under System variables, find Path, click Edit
  4. Click New and add: C:\EZ
  5. Click OK to save
