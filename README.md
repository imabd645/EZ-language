# EZ Language Interpreter (C++)

A lightweight **custom scripting language interpreter** written in **C++**, featuring dynamic typing, control flow, functions, arrays, and a clean, human-readable syntax.

This project demonstrates **compiler/interpreter fundamentals**, including lexical analysis, recursive-descent parsing, scope handling, and runtime execution.
