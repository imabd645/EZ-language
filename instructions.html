EZ Language Guide

Everything you need to master EZ programming

Introduction

EZ Language is a beginner-friendly programming language designed to make learning to code as easy as possible. It uses intuitive, human-readable keywords that feel natural to write and read.

Why EZ?

  • Natural Syntax - Keywords like when, repeat, and task read like English
  • No Semicolons - Focus on your logic, not punctuation
  • Readable Booleans - Use yes and no instead of true/false
  • Quick Learning Curve - Start coding in minutes, not hours

Installation

Step 1: Download EZ Interpreter

Download ez.exe from the downloads section.

Step 2: Create Installation Folder

Create a folder for EZ Language:

C:\EZ

Move ez.exe to this folder.

Step 3: Add to System PATH

  1. Press Win + R to open Run dialog
  2. Type sysdm.cpl and press Enter
  3. Click the Advanced tab
  4. Click Environment Variables
  5. Under System variables, find Path and click Edit
  6. Click New and add: C:\EZ
  7. Click OK on all dialogs

Step 4: Verify Installation

Open a new Command Prompt and type:

ez --version

Language Basics

Comments

// This is a single-line comment

Output

Use out to print to the console:

out "Hello, World!"
out 42
out "The answer is: " + 42

Input

Use in to get user input:

out "What is your name?"
in name
out "Hello, " + name + "!"

Variables & Data Types

Variables are created by assignment:

name = "Abdullah"
age = 25
pi = 3.14159
isStudent = yes
Type Example Description
String "Hello" Text in quotes
Number 42, 3.14 Integers and decimals
Boolean yes, no True/false values
Array [1, 2, 3] Collection of values

EZ Operator Description

Control Flow

If Statement (when)

age = 20

when age greaterthen 18 {
    out "You are an adult"
}

If-Else

age = 15

when age greaterthen 18 {
    out "Adult"
} other {
    out "Minor"
}

Loops

For Loop (repeat)

// Count from 1 to 5
repeat i = 1 to 5 {
    out i
}

While Loop (until)

x = 5
until x equal 0 {
    out x
    x = x - 1
}

Foreach Loop (get)

fruits = ["apple", "banana", "orange"]
get fruit in fruits {
    out fruit
}

Functions

Defining Functions (task)

task greet(name) {
    out "Hello, " + name + "!"
}

greet("World")

Return Values (give)

task add(a, b) {
    give a + b
}

result = add(5, 3)
out result  // 8

Arrays

// Creating arrays
numbers = [1, 2, 3, 4, 5]
names = ["Alice", "Bob", "Charlie"]

// Accessing elements
out numbers[0]  // 1

// Iterating
get num in numbers {
    out num * 2
}

Complete Examples

Fibonacci Sequence

task fib(n) {
    when n lessthen 2 {
        give n
    }
    give fib(n - 1) + fib(n - 2)
}

repeat i = 0 to 10 {
    out fib(i)
}

Simple Calculator

out "Enter first number:"
in num1
out "Enter second number:"
in num2
out "Enter operation (+, -, *, /):"
in op

when op equal "+" {
    out num1 + num2
} when op equal "-" {
    out num1 - num2
} when op equal "*" {
    out num1 * num2
} other {
    out num1 / num2
}
